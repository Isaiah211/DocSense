Various methods are available for segmentation, however, this article talks about robust Color based object segmentation. Create binary mask that separates blue T-shirt from rest. To find blue t-shirt in given image, I used OpenCV’s inRange method: Which takes color (or greyscale) image, lower & higher range value as its parameter and returns binary image, where pixel value set to 0 when input pixel doesn’t fall in specified range, otherwise pixel value set to 1.
